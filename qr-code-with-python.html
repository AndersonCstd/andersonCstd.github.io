How to generete Qr code with Python

Qr Codes have so many uses and one of them is to connect in Wi-Fi networks. With the code bellow you will be able to generete Qr Codes, attention to the dependencies:
pip install pyqrcode Also, is optional the use of the argparse that allows you to provide arguments in command-line, with no need to edit code!

Usage:
1 - Save the file as: gen_qr.py or whathaver you prefer;
2 - Run with: python gen_qr.py "SSID of the network (name)" "Security protocol" "Password"
For example: python gen_qr.py "my_wi-fi" "WPA2" "12345678"
